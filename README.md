## Getting Started

First, run the development server:

```bash
npm run dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
```
## Techstack: 
  ### NextJS, ShadcnUI, Tailwind, Typescript, React Hook Form, Zod Validator
  ### Mongoose, JWT, bcryptjs
